1. Installation Guide

To write the Celerity ISO image to a USB flash drive (minimum 4GB capacity), run the following command replacing `/dev/sdX` with your USB drive path:

dd if=celerity-os-24.04-desktop-amd64.iso of=/dev/sdX bs=4M status=progress && sync

Ensure **Secure Boot** is enabled in your BIOS; Celerity OS packages are fully cryptographically signed.
